## ⚠️ Key Consideration: State Ownership

Banco del Pacífico is **state-owned** via CFN (Corporación Financiera Nacional). This means:
- **Political appointment cycles** — Leadership may change with administrations
- **Public procurement rules** — SERCOP regulations apply
- **Budget constraints** — Government fiscal policy affects investment capacity
- **Longer decision cycles** — Bureaucratic approval processes
- **Policy alignment** — Technology investments may need to align with government priorities

**Critical:** Research current administration's stance on public sector modernization.

```
┌─────────────────────────────────────────────────────────┐
│              PUBLIC PROCUREMENT CYCLE                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. NEED IDENTIFICATION                                 │
│     └── Internal technical team defines requirements    │
│                                                         │
│  2. BUDGET ALLOCATION                                   │
│     └── CFO + CFN approval for major investments        │
│                                                         │
│  3. PROCUREMENT PLANNING                                │
│     └── Annual procurement plan (PAC) publication       │
│                                                         │
│  4. TENDER PROCESS                                      │
│     └── SERCOP portal publication                       │
│     └── Technical specifications + evaluation criteria  │
│                                                         │
│  5. EVALUATION & AWARD                                  │
│     └── Technical + economic evaluation                 │
│     └── Public notification                             │
│                                                         │
│  6. CONTRACT & EXECUTION                                │
│     └── Formal contracting                              │
│     └── Implementation                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Key Procurement Considerations

1. **Annual Procurement Plan (PAC)** — Check SERCOP for Banco del Pacífico's published plan
2. **Threshold Rules** — Different processes for different contract values
3. **Technical Specifications** — CIO/Infrastructure team writes these; influence opportunity
4. **Qualification Requirements** — May require local presence or partnerships
5. **Timeline** — Public procurement adds 3-6 months minimum to sales cycle
