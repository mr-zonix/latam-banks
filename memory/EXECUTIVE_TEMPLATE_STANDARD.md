# Executive Page Standard Template

Based on Banco Pichincha (the gold standard)

## REQUIRED SECTIONS

1. **Verified Source Badge** — Link to official bank "Alta Dirección" page
2. **Update Badge** (if applicable) — Recent CEO changes with source
3. **Org Chart Visual** — Real names in hierarchy boxes
4. **6 Executive Profiles:**
   - Chairman (if applicable)
   - CEO / Gerente General
   - VP Tech / CTO / CIO
   - VP Transformación / Digital
   - CISO / Security
   - CDO / Data Leader (or other relevant)

## EACH EXECUTIVE PROFILE MUST HAVE:
- Real name (not "To Verify" placeholder)
- Photo or initials placeholder
- Role + tenure
- Background/experience bullets
- Strategic focus bullets
- Engagement strategy box
- LinkedIn search link

## RESEARCH METHODOLOGY:
1. **Official website** → "Alta Dirección" / "Gobierno Corporativo" / "Equipo Directivo"
2. **Annual reports** → Usually list executives
3. **News search** → Recent appointments/changes
4. **LinkedIn company page** → People section

## STRUCTURE CONSISTENCY:
- Same CSS/styling as Pichincha
- Same sidebar with 8 items
- Same color coding (Chairman=orange, CEO=blue, Tech=green, Digital=purple, Security=red)
- Same engagement strategy boxes
- v4.3 • Apr 2026 version

## DO NOT:
- Use "To Verify" placeholders — research until you find the name
- Skip the engagement strategy boxes
- Use different styling than Pichincha
- Leave empty sections
