# 🏦 TEMPLATE — Perfil de Banco (Estructura Aprobada)

Este es el template estándar para perfiles de banco. Basado en Banco de la Nación que a Zuni le gustó.

---

## Estructura de la Página HTML

### 1. Header
- Badge de tipo (Estatal, Privado, etc.)
- Nombre del banco
- Subtítulo descriptivo

### 2. Datos Clave (Métricas)
```
| Activos | Agencias | Cobertura | Empleados | Ranking |
```

### 3. Warning Box (si aplica)
- Consideraciones especiales de venta
- Proceso de compra
- Ciclos
- Pitch ganador

### 4. Perfil Institucional
- Tabla con datos básicos
- Mandato/Funciones principales

### 5. Ejecutivos Clave (VERIFICADOS)
- Contactos prioritarios para TI (destacados)
- Directorio
- Otras gerencias
- Incluir: nombre, cargo, teléfono, fuente de verificación

### 6. Oportunidades de Infraestructura
- Grid de 4 cards con prioridades
- Cada card: Oportunidad, Pitch, Soluciones

### 7. Análisis por Capas ⚠️ INCLUIR PROVEEDORES
```
| Capa | Estado | Proveedores | Oportunidad | Prioridad |
|------|--------|-------------|-------------|-----------|
| Compute | ... | IBM, VMware | ... | 🟢 |
| Storage | ... | Dell EMC | ... | 🟢 |
| Network | ... | Cisco, Claro | ... | 🟢 |
| Core | ... | COBIS, Temenos | ... | 🟡 |
| Digital | ... | Backbase | ... | 🟡 |
| Data/AI | ... | Oracle, SAS | ... | 🟢 |
```

### 8. Estrategia de Acercamiento
- Fases: Identificación → Acercamiento → Técnico → Cierre

### 9. Enlaces Útiles
- Web oficial
- Directorio funcionarios
- Licitaciones
- Regulador

## En el Dashboard del País (index.html)

```
| Banco | Activos | Agencias | ROE | CIR | Crecimiento | Oportunidad |
```
