BBVA Group runs a global cloud-first strategy mandated from Spain. AWS is the primary cloud provider globally, with Google Cloud for specific workloads. ⚠️ Global Constraint: BBVA Spain sets cloud strategy. Local infra purchases follow global procurement agreements.
💾 Layer 2: Enterprise Storage LOW OPP (Global)
AWS S3/EBS NetApp (Legacy) Dell EMC Pure Storage

Storage follows cloud-first mandate. New applications use AWS native storage (S3, EBS, EFS). Legacy on-prem storage likely NetApp or Dell EMC, being phased out. Data growth driven by regulatory requirements and digital banking expansion. SBS Peru requires 10-year data retention.

🌐 Layer 3: Networking MED OPP
Cisco (Core & DC) Fortinet Security Palo Alto (Perimeter) AWS Direct Connect

270+ branches require robust WAN infrastructure. SD-WAN transformation likely completed or in progress. Cisco dominates core and data center networking. Hybrid connectivity via AWS Direct Connect to cloud workloads. Security stack includes next-gen firewalls at perimeter and branch locations. Network is area with SOME local autonomy due to regulatory requirements.

🏛️ Layer 4: Core Banking LOW OPP (Global)
ALNOVA (BBVA Proprietary) API Connect Microservices Layer

BBVA developed ALNOVA — their proprietary core banking platform deployed across all subsidiaries. This is a GLOBAL DECISION with zero local autonomy. ALNOVA is modern, cloud-native, API-first architecture. Peru was one of the first markets to migrate to ALNOVA (2017-2019). No core banking opportunity here — fully controlled by Spain.

🚫 No Opportunity: ALNOVA is proprietary. Core modernization is complete. No vendor opportunity.
📱 Layer 5: Digital Banking Platform MED OPP
BBVA App (Global Platform) API Market Kubernetes/EKS Open Banking APIs

BBVA Perú app is based on global BBVA platform — #1 rated banking app in multiple markets. Strong digital onboarding: 100% digital account opening since 2019. Open Banking leader in Peru — first to launch API Market. ~78% of customers are digital active. Mobile-first strategy with 4.8M digital customers. Some local customization allowed for regulatory compliance and local market features.

📊 Layer 6: Data, Analytics & AI MED OPP
BBVA AI Factory AWS SageMaker Snowflake Tableau/Power BI

BBVA has a global AI Factory that develops ML models for all subsidiaries. Peru implements: fraud detection AI, credit scoring models, personalization engines, chatbots. Data platform likely Snowflake or AWS-native analytics. Strong data governance due to global standards. Local team can customize models but core AI/ML capabilities come from global. NPL increase (4.8%) creates opportunity for advanced credit risk analytics.

💡 Opportunity: Credit risk analytics, local market intelligence, regulatory reporting automation.
